欧拉函数

定义：设 nnn 是一个正整数，欧拉函数 φ(n)\varphi(n)φ(n) 定义为不超过 nnn 且与 nnn 互素的正整数的个数。

欧拉函数的性质

具体证明请参考《初等数论及其应用》第6，7章。

  1. 欧拉函数是积性函数。
  2. n=∑d∣nφ(d)n = \sum_{d \mid n}{\varphi(d)}n=∑d∣n​φ(d)。
  3. 若 n=pkn = p^kn=pk，其中 ppp 是质数，那么 φ(n)=pk−pk−1\varphi(n) = p^k - p^{k - 1}φ(n)=pk−pk−1。
  4. 由唯一分解定理，设 n=∏i=1spikin = \prod_{i=1}^{s}p_i^{k_i}n=∏i=1s​piki​​，其中 pip_ipi​ 是质数，有 φ(n)=n×∏i=1spi−1pi\varphi(n) = n \times \prod_{i = 1}^s{\dfrac{p_i - 1}{p_i}}φ(n)=n×∏i=1s​pi​pi​−1​。
  5. ∀n, n>1\forall n,\ n>1∀n, n>1 ，1,2,⋯ ,n1,2,\cdots,n1,2,⋯,n 中与 nnn 互质的数的和为 n×φ(n)2n\times \dfrac{\varphi(n)}{2}n×2φ(n)​ 。

分解质因数求欧拉函数

如果只要求一个数的欧拉函数值，那么直接根据定义质因数分解的同时求就好了。这个过程可以用 Pollard Rho 算法优化。

筛法求欧拉函数

注意到在线性筛中，每一个合数都是被最小的质因子筛掉。比如设 p1p_1p1​ 是 nnn 的最小质因子，n′=np1n' = \dfrac{n}{p_1}n′=p1​n​，那么线性筛的过程中 nnn 通过 n′×p1n' \times p_1n′×p1​ 筛掉。
观察线性筛的过程，我们还需要处理两个部分，下面对 n′ mod p1n' \bmod p_1n′modp1​ 分情况讨论。
如果 n′ mod p1=0n' \bmod p_1 = 0n′modp1​=0，那么 n′n'n′ 包含了 nnn 的所有质因子。
