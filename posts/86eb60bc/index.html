题目

P4626 一道水题 II 。

求 1~n 的最小公倍数。 n≤1e8n\le 1e8n≤1e8

思路

欧拉筛 1e8，对于小于等于 n 的每个素数，找其小于等于 n 的最大次幂的值。开 O2 即可 AC。

然后 开始优化 ！

  • 当 n 较大时，使用欧拉筛可以先把偶数滤掉。平均每个点快 60 ms。

  • 在欧拉筛中，for(int j = 1; p[j] <= n / i; j++) 这一步为了防止溢出，大家通常会用除法，但通过控制变量法，我发现用 for(int j = 1; 1ll * p[j] * i <= n; j++) 会更快！这也又一次说明了除法是有多么慢！

  • 最后，若一个素数 p 大于 n\sqrt{n}n​ ，那么小于等于 n 时，p的最大次幂为 1。所以可以直接 ans = ans * p % mod 。这个其实影响不大，因为用 while 也就多 1 次乘法。
